#Руководство пользователя

Данная программа позволяет запускать сборку проектов на удаленном сервере. Поддерживается сборка 
следующих типов проектов:
*Make Sources
*Build PC
*Build Release
*Build Full
*Build Other

##Подключение к серверу 
Для начала работы необходимо подключиться к запущенному серверу на удаленной машине.
Для этого:

*Нажмите кнопку "Connection"
*Нажмите кнопку "Open"

Далее в появившемся окне необходимо ввести Server IP удаленной машины, на которой запущен
сервер и Port:

*Введите Server IP
*Введите Port
*Нажмите кнопку "Open"

После подключения к серверу появится список в окне с доступными проектами, которые данный 
сервер поддерживает для сборки. Справа от проекта цветными кружочками будет отображаться 
текущее состояние всех типов сборки данного проекта.
Возможны следующие состояния:
*Зеленый:  проект был успешно построен
*Желтый: проект строится в текущий момент
*Красный: либо проект еще не собирался с момента запуска сервера, либо при последней сборке 
произошла ошибка

Цифрами в кружочках будет отображаться текущая позиция проектов в очереди на построение.

В Tooltip'e красного кружочка пользователь может увидеть причину ошибки построения.
В Tooltip'e зеленого и желтого кружочков пользователь увидит версию проекта, которая была собрана 
или собирается в данный момент соответственно.

##Инициирование сборки пользователем
Для того, чтобы инициировать сборку пользователю необходимо в GroupBox'e "Version" указать требуемую
версию сборки проекта, затем нажать на одну из кнопок, которая будет инициировать сборку:
*Make Sources
*Build PC
*Build Release
*Build Full
*Build Other

После чего в соответствующем кружочке появится номер позиции текущего пользователя в очереди проектов.
Если же до этого момента очередь была пуста, то требуемая сборка начнется незамедлительно.
В момент сборки пользователь в окне "Logs" может по соответствующим log'ам отследить состояние сборки
своего проекта.

##Отмена сборки
Для отмена построения пользователю необходимо выбрать требуемый проект, нажать кнопку "Cancel", после чего
все типы построений по данному проекту, которые инициировал данный пользователь будут отменены.
