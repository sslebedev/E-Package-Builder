# Руководство администратора сервера

Данная программа позволяет менять настройки сборки проектов на удаленном сервере.

## Подключение к серверу 
Для начала работы необходимо подключиться к запущенному серверу на удаленной машине.
Для этого:

* Нажмите кнопку "Connection"
* Нажмите кнопку "Open"

Далее в появившемся окне необходимо ввести Server IP удаленной машины, на которой запущен
сервер и Port:

* Введите Server IP
* Введите Port
* Нажмите кнопку "Open"

После подключения к серверу появится список в окне с доступными проектами, которые данный 
сервер поддерживает для сборки.


## Получение настроек сборки

* Кликните по названию одного из проектов

В полях для редактирования путей к элементам сборки появятся пути до соответствующих элементов.

## Редактирование настроек сборки

* Выберите соответствующий путь в поле редактирования.
* Отредактируйте

Для сброса редактирования одного из полей редактирования нажмите кнопку "Reset" для соответствующего
поля.
Для сохранения результатов редактирования нажмите кнопку "Save".
Для отмены редактирования настроек текущего проекта нажмите кнопку "Cancel"

## Меню программы
* "Connection" требуется для подключения к серверу см. выше.
* "About". В этом разделе есть элемент "Program", нажав на который Вы увидите данную документацию.
